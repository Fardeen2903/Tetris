# Tetris
Simple tetris clone using pygame

A classic Tetris game implemented in Python using `pygame`. The game features the typical Tetris gameplay with a falling block system, rotation, and collision detection. The game includes a score system, level progression, and high score tracking.

## Features

- **Piece Rotation**: Rotate tetrominoes using the spacebar.
- **Controls**: 
  - **Arrow Keys**: Move the piece left, right, or down.
  - **Spacebar**: Rotate the piece.
  - **F Key**: Toggle fullscreen mode.
- **Scoring**: Clear lines to increase your score. The game speed increases as you level up.
- **High Score**: Save and display the highest score achieved.

## Requirements

- Python 3.x
- pygame (Install with `pip install pygame`)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Fardeen2903/Tetris.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Tetris
   ```

3. Install dependencies:
   ```bash
   pip install pygame
   ```

## Usage

To start the game, simply run the `main.py` file:

```bash
python main.py
```

Enjoy the game!
